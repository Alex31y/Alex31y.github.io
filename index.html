Thesis on Data Analysis
and Statistical Inference

Analysis of a dataset containing 10000 istances of Test Cases regarding open source projects.
Im looking for any or some feature describing the generic test case which has a correlation with overall code coverage achieved by the single test case. Finally i developed a Random Tree Forest regression model able to predict the dependent variable with a Mean Absolute Error of 1.5 ; this implies that if the test case given in input to the model achieves a code line coverage of let's say 10, the predicted value is between 8.5 and 11.5
